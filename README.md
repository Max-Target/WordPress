##Installation
 - Выберите Плагины -> Добавить новый.
 - В строке поиска введите MaxTarget.
 - Установите плагин *MaxTarget* и активируйте его.
 - Введите Еmail и полученный на сайте http://maxtarget.ru Ключ API.
